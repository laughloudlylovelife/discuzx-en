User interface

This part of the interface is responsible for user registration, login, enabling users to form a unified management.

Short message interface

This part of the interface is responsible for the short message transmission between users, across applications to send short messages between each other. "Short Message Center" screen interface allows the application of only one function interface that has a short message function, for advanced users UCenter transceiver also provides a rich user interface function for open Send PM system.

This part of the user interfaces for information sharing friend. Friends updated information on an application be able to see other applications.
