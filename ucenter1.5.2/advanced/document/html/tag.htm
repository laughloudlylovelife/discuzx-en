Get tag data

array uc_tag_get(string tagname [, array nums])

Function parameters
parameter meaning
string tagname Tag name
array nums Specify how many of each application to return data, the array format is as follows:
KEY VALUE
Application ID article number to return data
Return Value
value meaning
array Tag data, a single tag array structure see Schedule

The center to the user interface to initiate a request to require all applications for users to center the label data. If you specify totalnum, it will be in accordance with the User Center tab to set the number in the appropriate proportion to the number of entries to return, if not specified, each application will return all of the 10 data.
Get tag data sample (PHP)
$nums = array(
	1 => 3,//ID for the application of a return to three data
	2 => 5,//ID for the two applications back to five data
);

$datalist = uc_tag_get($_GET['tagname'], $nums);

foreach($datalist as $tag) {
	echo "<b>$tag[type]:</b><br />";
	foreach($tag['data'] as $data) {
		echo "<a href='$data[url]'>$data[subject]</a><br />$data[extra]";
	}
}

Schedule: single tag array structure

array ['data'] Labels array
string ['url']URL
string ['subject']Title
string ['extra']Extended Data
integer ['type']Application type, value DISCUZ, SUPESITE, XSPACE, SUPEV, ECSHOP, OTHER
