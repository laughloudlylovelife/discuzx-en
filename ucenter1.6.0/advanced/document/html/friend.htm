Add Friends

bool uc_friend_add(integer uid , integer friendid [, string comment])

function parameters
parameter meaning
integer uid User ID
integer friendid Friend User ID ID
string comment Notes, may be empty
return value
value meaning
bool 1: success
0: fail

The function interface is used to add the friend uid to the friend list.
Add Friend example (PHP)
$friendid = uc_get_user($_POST['friend']);
if(uc_friend_add($uid, $friendid[0])) {
	echo 'Add completed';
}

Remove Friend

integer uc_friend_delete(integer uid , array friendids)

function parameters
parameter meaning
integer uid User ID
array friendids Friend User ID array
return value
value meaning
integer deleted a few friends

The function interface is used to delete the specified user's friends. friendids parameter is an array of free Key type, such as "array (1, 2, 3, 4, 5 ...)", array values are on behalf of friends of each user ID.
Remove Friend example (PHP)
$if(!empty($_POST['delete']) && is_array($_POST['delete'])) {
	$num = uc_friend_delete($uid, $_POST['delete']);
	echo "Deleted $num friend(s)";
}

Get friends

integer uc_friend_totalnum(integer uid [, integer direction])

function parameters
parameter meaning
integer uid User ID
integer direction 0: (Default value) specifies the user's entire friends
1: is positive, add the specified user's friends, but no one was added
2: back to specify which users add the user to be friends, but no one was added
3: two-way, mutual Add as Friend
return value
value meaning
integer friends number

This interface function is used to return the specified user's friends number.
Get friends example (PHP)
$num = uc_friend_totalnum($uid);
echo "You have $num friend(s)";
