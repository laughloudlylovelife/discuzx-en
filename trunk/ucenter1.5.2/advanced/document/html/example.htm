Example program

In order to facilitate the development, we put all the UCenter interface function together to produce a sample program. To run this example program, you need to do to work as follows:

1. sample program does not contain a default client directory, if you want to run this sample program, copy the client installation package UCenter example programs to the root directory.
2. in UCenter add an application, set the example to run the URL and other information, communication keys are free to fill out, such as "123456."
3. open the sample program to connect UCenter config.inc.php to set examples of the way, set the database connection parameters. "UC_KEY" complete set of communication in UCenter key (the example of the "12345"), "UC_APPID" fill out the application just to add ID.

ucexample_1.php for the sample program without the user's master file table
ucexample_2.php users to have their own examples of program master file table

Specific directory structure, please see table below:
client/ UCenter client's file directory, the default is not included in this directory.
api/ API interface directory
code/ Example code program segment directory
code/login_nodb.php ucexample_1.php user login sample code, including logon authentication, synchronization log on the example of the code. You know there is no user table can also be integrated UCenter.
code/login_db.php ucexample_2.php User login sample code, including logon authentication, login and user activated simultaneously sample code.
code/register_nodb.php ucexample_1.php User registration example code, contains an example of registration information verification code.
code/register_db.php ucexample_2.php Example of a user registration code, including registration information verified, the user activate the judge's example code.
code/pmwin.php Short message center directly into the sample code.
code/pmlist.php Examples of self-made short message platform code, including access to the list of short message, content, and sample code to send a short message.
code/friend.php Friend list example code
code/avatar.php Avatar view/edit sample code
include/ Library files
view/ Template Directory
