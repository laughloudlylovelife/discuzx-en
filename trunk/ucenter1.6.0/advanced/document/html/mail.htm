add messages to the queue

mixed uc_mail_queue(stringuids, stringemails, stringsubject, stringmessage[, mailfrommail, stringcharset, booleanhtmlon, integerlevel]);

function parameters
parameters meaning
string uids multile user ID, separated with a comma (,)
string emails multiplt target email, seperated with a comma (,)
string subject message title
string message message content
mail frommail sender, optional parameters, default is empty, uc background set the source e-mail address as the sender
string charset message character set, optional parameters, default is gbk
boolean htmlon whether it is html format e-mail, optional parameter, defaults to FALSE, the text message
integer level message-level, optional parameters, default is 1, the priority to send large numbers, when the value is 0 immediately send, the message would not join the queue
return value
value meaning
mixed false: failure: failure to enter the queue, or send failure
integer: success: access to the message queue id, when the level is 0, then return 1

This function is used to e-mail to e-mail queue, or directly send messages (when the level is 0)
message queue using example (PHP)
$res = uc_mail_queue($uid, '', $_POST['subject'], $_POST['message']);
if(empty($res)) {
	echo 'access to the message queue failed';
} else {
	echo 'access to the message queue successfully';
}
